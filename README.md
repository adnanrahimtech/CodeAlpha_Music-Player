# 🎵 CodeAlpha Music Player

A modern, feature-rich web-based music player built with vanilla JavaScript, HTML, and CSS. Perfect for playing your favorite tracks with a sleek and intuitive user interface.

## ✨ Features

- **Play/Pause Control** - Easy control over playback
- **Track Navigation** - Skip to next or previous tracks
- **Volume Control** - Adjust volume with an intuitive slider
- **Progress Bar** - See and control track progress
- **Playlist Support** - Manage multiple tracks
- **Time Display** - Current time and total duration
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Modern UI** - Clean, user-friendly interface with smooth animations

## 🛠️ Tech Stack

- **JavaScript** (54.8%) - Core application logic and interactivity
- **CSS** (32.6%) - Styling and animations
- **HTML** (12.6%) - Structure and markup

## 📦 Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No external dependencies required

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/adnanrahimtech/CodeAlpha_Music-Player.git
   cd CodeAlpha_Music-Player
   ```

2. **Open in your browser**
   - Simply open the `index.html` file in your web browser
   - Or use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     ```

3. **Access the application**
   - Open `http://localhost:8000` in your browser

## 🎶 Usage

### Basic Controls

1. **Add Music** - Load your audio files into the player
2. **Play/Pause** - Click the play button to start or pause playback
3. **Skip Tracks** - Use next/previous buttons to navigate through songs
4. **Adjust Volume** - Use the volume slider to control audio level
5. **Seek Position** - Click on the progress bar to jump to a specific time

### Keyboard Shortcuts (if implemented)

- `Space` - Play/Pause
- `→` - Next track
- `←` - Previous track
- `↑` - Increase volume
- `↓` - Decrease volume
